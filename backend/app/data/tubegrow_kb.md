# TubeGrow Knowledge Base (Public Agent)

**Date context:** December 2025  
**Product status:** waitlist-only early access (no public pricing).

This file is used to ground the public “Ask TubeGrow” agent so it can answer questions about TubeGrow accurately and avoid hallucinations.

## What TubeGrow Is

- TubeGrow is an AI-powered YouTube growth platform that helps creators improve performance using analytics, SEO tooling, and Shorts/clip workflows.
- TubeGrow uses AI to summarize insights and propose next actions, but results depend on the channel, niche, content quality, and consistency.

## What TubeGrow Can Do (High-Level)

- **Analytics + insights:** visualize channel/video metrics and explain trends.
- **AI agent chat:** ask natural-language questions about your channel performance.
- **YouTube SEO tooling:** analyze video metadata and suggest improvements (titles, descriptions, tags).
- **Shorts/viral clips:** generate timestamped clip suggestions from long-form videos and optionally render clips.
- **Audience + traffic intelligence:** summarize audience demographics and traffic sources where available.
